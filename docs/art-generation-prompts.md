## Quick Generation Guide

### Using These Prompts

1. **Choose your image generation tool**: Midjourney, DALL-E, Stable Diffusion, etc.
2. **Copy the prompt** for the asset you want to create
3. **Adjust if needed** based on your tool's requirements
4. **Generate the image**
5. **Save to the specified target file location**
6. **Verify quality** - check size, colors, child-friendly aesthetic

### Batch Generation Tips

For efficient generation of multiple assets:

1. **Group by type**: Generate all weapons, then shields, then backgrounds
2. **Consistent style**: Use same generation tool for related assets
3. **Quality check**: Ensure consistent visual style across similar assets
4. **Child-safety review**: Verify all images are age-appropriate

### Post-Generation Steps

After generating images:

1. **Resize** to exact specifications if needed
2. **Optimize** file size (PNG compression)
3. **Rename** following naming conventions
4. **Place** in correct directory
5. **Update** art-guide.md with image references
