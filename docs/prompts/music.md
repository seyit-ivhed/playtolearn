# Music Prompts

This file contains prompts used for generating music in Udio for Math Quest Adventures.

### Music Prompt for Main Theme / Chronicles

**Description**: Main background music for the game launch and Chronicles page. Revised to be more upbeat, optimistic, and energetic while retaining the magical adventure feel.

**Udio Prompt**:
> Upbeat orchestral fantasy adventure, bright and cheerful strings, spirited woodwinds melody, heroic brass swells, energetic styling, uplifting, optimistic, magical, 105 BPM, loopable, high quality

**Musical Elements**:
- **Genre**: Upbeat Orchestral Fantasy
- **Key Instruments**: Bright Strings, Woodwinds (Flute/Oboe), Brass
- **BPM**: ~105 BPM
